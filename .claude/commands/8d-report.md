## 입력
- `$ARGUMENTS`: 이슈 설명 (예: `ESD 시험 중 BCM 리셋 현상 발생`)

## 작업 절차 (1차: au-test-system 앱 `/ncr`)

au-test-system 앱이 M05(NCR 관리)에서 8D 필드(D1~D8 탭)·댓글·첨부를 이미 지원한다. 아래 8D 구조로 초안을 작성한 뒤 `POST /ncr/`(또는 프론트엔드 `/ncr` 등록 폼, `NCR-YYYY-NNN` 자동 채번)로 등록하는 것이 기본. HKMC 제출용 Word/PPT 문서가 별도로 필요할 때만 `05_NCR_8D/NCR_8D_보고서_템플릿.docx`·`8D_보고서_PPT_템플릿.pptx` 서식 참조.
